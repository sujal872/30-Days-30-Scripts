# 💰 Simple Expense Tracker (Python CLI)

A simple **Command Line Expense Tracker** built using **Python**.
This project helps users **record, manage, and analyze daily expenses** using a CSV file for storage.

It is a beginner-friendly project that demonstrates real-world Python concepts like **file handling, date management, and data aggregation**.

---

# 🚀 Features

✔ Add Expense
✔ View All Expenses
✔ Monthly Expense Summary
✔ Category-wise Expense Analysis
✔ Edit Existing Expense
✔ Data stored in CSV file
🔥 Export summary report (Bonus Feature)

# 📝 Example CSV Data

Your `Expenses.csv` file will store data like this:

```
2026-03-01,Food,Burger,120
2026-03-01,Bills,Electricity,800
2026-03-02,Travel,Metro,50
2026-03-03,Food,Pizza,300
2026-03-04,Shopping,Tshirt,600
```

Columns format:

```
Date,Category,Item,Expense
```
