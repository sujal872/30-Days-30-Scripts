# 📝 To-Do List (File Based) - Python Project

A simple File-Based To-Do List application built using Python.

## 🚀 Features

- Add Task
- View All Tasks
- Delete Task
- Mark Task as Completed
- Add Due Date
- File Storage (Persistent Data)
- Proper CRUD Operations

---

## 📂 Project Structure

- 📁 Day-03-To-Do-List
- ├── todo.py
- ├── to-do-list.txt
- └── README.md


---

## 🛠 Concepts Used

- Python Lists
- File Handling (Read / Write / Append)
- CRUD Operations
- Exception Handling
- String Manipulation

---

## 📌 How It Works

- Tasks are stored in a file: to-do-list.txt

- Format: Task Name | Due Date | Status

- Example: Complete Python Project|2026-03-10|Pending

## ▶️ How To Run

```bash
python todo.py

📸 Example Output
```
