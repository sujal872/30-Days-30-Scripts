# 🎯 Number Guessing Game (Python CLI)

A fun and interactive Command-Line Number Guessing Game built using Python.

This project demonstrates core Python concepts including:
- Functions
- Loops
- Conditional statements
- Exception handling
- File handling
- Game logic implementation

---

## 🚀 Features

✔ Difficulty Levels (Easy / Medium / Hard)  
✔ Maximum 7 Attempts per Game  
✔ High Score Tracking (Stored in File)  
✔ Separate High Score for Each Difficulty  
✔ Replay Option  
✔ Input Validation  
✔ Clean & Structured Code  

---

## 🎮 Difficulty Levels

| Level   | Number Range |
|----------|-------------|
| Easy     | 1 – 50      |
| Medium   | 1 – 100     |
| Hard     | 1 – 200     |

Each level maintains its own high score.

---

## 🏆 High Score System

- High scores are saved in `score.txt`
- If a player beats the previous high score, it automatically updates
- Separate tracking for:
  - easy
  - medium
  - hard

Example `score.txt`:
easy:3
medium:5
hard:6
