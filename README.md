# 🚀 30 Days – 30 Scripts Python

A challenge to build **30 Python projects in 30 days**, ranging from **basic automation scripts to security-focused tools**.

The goal of this challenge is to improve:

* 🧠 Problem Solving
* 🐍 Python Programming
* 🔐 Cybersecurity Concepts
* ⚙️ Automation Skills

Each day includes a **small but practical script** that demonstrates a specific concept such as file handling, encryption, networking, or OSINT techniques.

---

# 🎯 Challenge Goals

* Build **30 real-world Python scripts**
* Practice **Python fundamentals to advanced concepts**
* Learn **security-related scripting**
* Create a **strong GitHub portfolio**

---

# 🛠️ Technologies Used

* Python 3
* JSON Storage
* File Handling
* CLI Tools
* Encryption Basics
* Networking
* Security Automation
