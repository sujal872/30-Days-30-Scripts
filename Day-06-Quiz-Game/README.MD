# 🧠 Python Quiz Game (CLI)

A simple **Command Line Quiz Game** built with Python.
This project asks multiple Python-related questions, tracks the user's score, calculates the percentage, and assigns a grade based on performance.

This project is part of my **#30Days30Scripts Python Challenge**.

---

## 📌 Features

* 📝 Multiple Quiz Questions
* 🎯 Score Tracking
* 📊 Percentage Calculation
* 🏆 Grade System (A / B / C / D)
* ⏱️ Quiz Timer
* ❌ Correct & Incorrect Answer Feedback
* 🖥️ Clean Command Line Interface

---

## 🛠️ Technologies Used

* Python 3
* Time Module
* CLI (Command Line Interface)

## 🏆 Grading System

| Percentage | Grade |
| ---------- | ----- |
| 90% - 100% | A     |
| 70% - 89%  | B     |
| 40% - 69%  | C     |
| Below 40%  | D     |
