# 📒 Contacts Book (Python)

A simple **CLI-based Contacts Book** built in Python that allows users to store and manage contacts using a **JSON file database**.

This project is part of my **30 Days – 30 Scripts of Python Challenge**.

---

## 🚀 Features

✔ Add Contact  
✔ Search Contact  
✔ Update Contact  
✔ Delete Contact  
✔ View All Contacts  
✔ Store contacts in JSON  
✔ Phone number validation  
✔ Email validation  

---

## 🧠 Concepts Used

- Python Dictionary
- JSON File Handling
- Input Validation
- Regex (Email Validation)
- CLI Menu System
