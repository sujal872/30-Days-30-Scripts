# 🔐 Day 01 - Advanced Password Tool

A Python-based command line Password Tool that includes:

- 🔎 Password Strength Checker
- 🔑 Custom Password Generator
- 🎯 User-based dynamic password creation
- 🔥 Generates 4 strong randomized passwords

This project is part of my **30 Days 30 Scripts Python Challenge**.

---

## 🚀 Features

### 1️⃣ Password Strength Checker
Checks password strength based on:
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters
- Minimum length validation

Strength Levels:
- Weak ❌
- Medium ⚠
- Strong ✅
- Very Strong 🔥

---

### 2️⃣ Password Generator

User can:
- Select password length (8–16)
- Add custom name/word
- Add favorite numbers
- Add custom special characters

Tool generates:
- 4 randomized strong passwords
- Mixed shuffled secure combinations

---

## 🛠 Technologies Used

- Python 3
- random module
- string module

---

## 📌 How to Run

1. Make sure Python 3 is installed
2. Open terminal in project folder
3. Run:

```bash
python password_tool.py
```
